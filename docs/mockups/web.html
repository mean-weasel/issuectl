Dashboard/seatify/#515

End-to-end guest data pipeline: RSVP quizzes, planner input, and optimization integration

Build a complete pipeline connecting guest RSVP responses to the seating optimization engine.

The flow should be:

  1. Guest fills out RSVP quiz (dietary prefs, plus-one info, table preferences, relationship tags)
  2. Planner reviews and augments guest data (add notes, override prefs, flag VIPs)
  3. Optimization engine ingests the enriched guest data alongside table constraints
  4. Results feed back into the canvas view

Key files:
src/lib/optimization/engine.ts
src/components/rsvp/QuizForm.tsx
src/app/api/guests/route.ts

The quiz should support custom questions per event. Planner input should be a spreadsheet-like table editor, not a form.
